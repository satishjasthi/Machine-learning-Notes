*To get the the column names of the pandas data frame use: dataframename.columns.values

*To access any column from a dataframe use:
dataframename.columnname

*To check wether a column has NaN or None value use
pd.isnull(object)

ex:pd.isnull(dataframename.colName)

where object --> column name or list
pd.isnull() returns a list of True and False

use pd.notnull to find the non null values

*to get the information like the datatype of columns ,number of non null entries in dataframe,name of each column of dataframe use
dataframename.info()

*to get the first few entries of dataframe use 
dataframename.head()
and to get few entries from bottom side use
dataframname.tail()

*to get the descriptive stats of the dataframe use
dataframename.describe()

*to create a list of equal spaced numbers between two numbers use
np.linspace(lowerlimit,upperlimit,number_of_parts)

Ex: np.linspace(-5,5,100)#divide the space beteween -5 and  into 100 parts

*given the descriptive stats, we can draw the normal distribution like

import matplotlib.pyplot as plt
import numpy as np
import matplotlib.mlab as mlab
import math

mu = 0
variance = 1
sigma = math.sqrt(variance)
x = np.linspace(-3, 3, 100)
plt.plot(x,mlab.normpdf(x, mu, sigma))

plt.show()

by using describe() like this and observing the values of the features in between these percentiles we'can gain a lot of information about feature's behaviour
